# 작업 방식 (중요)

이 저장소에서 작업할 때 아래 기본 원칙을 지켜줘:

1. **코드/설정을 바꾸기 전에 먼저 의견을 제시해줘.**
   내가 "실행해줘", "만들어줘"처럼 명확히 지시하기 전까지는
   접근 방식과 트레이드오프만 설명하고 실제 변경은 하지 마.

2. **커밋·푸시하기 전에 스크린샷이 필요한지 먼저 물어봐줘.**
   내가 원하면 그때 목 서버 + Playwright로 캡처해서 보여주면 돼.
   매번 자동으로 스크린샷을 찍을 필요는 없어.
   그리고 내가 "푸시해줘"라고 말하기 전까지는 커밋/푸시하지 마.
